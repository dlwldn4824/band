# 구글 드라이브 링크 설정 가이드

## 구글 드라이브 폴더 공유 링크 생성 방법

1. **구글 드라이브에서 폴더 생성**
   - 구글 드라이브 접속
   - 새 폴더 생성 (예: "밴드 공연 사진")

2. **폴더 공유 설정**
   - 폴더 우클릭 → "공유" 선택
   - "링크가 있는 모든 사용자" 선택
   - "뷰어" 또는 "편집자" 권한 설정
   - 링크 복사

3. **환경변수 설정**
   - `.env` 파일에 추가:
   ```
   VITE_GOOGLE_DRIVE_LINK=https://drive.google.com/drive/folders/YOUR_FOLDER_ID
   ```
   - `YOUR_FOLDER_ID`를 실제 폴더 ID로 교체

4. **또는 코드에서 직접 수정**
   - `src/pages/Chat.tsx` 파일의 `GOOGLE_DRIVE_LINK` 변수 수정

## 링크 형식

구글 드라이브 폴더 링크는 다음과 같은 형식입니다:
```
https://drive.google.com/drive/folders/FOLDER_ID
```

## 참고사항

- 폴더를 "링크가 있는 모든 사용자"로 공유해야 누구나 접근 가능합니다
- "편집자" 권한으로 설정하면 사용자가 직접 사진을 업로드할 수 있습니다
- "뷰어" 권한으로 설정하면 사진만 볼 수 있습니다
